# Smart Finance Tracker & Advisor (FINTRACK PLUS)

FINTRACK PLUS is a personal finance management application built with **Python** that helps users organize and track their finances in one place. It enables users to record income, expenses, savings, budgets, and debts while providing an overview of their financial health through summaries and basic AI-powered financial guidance.

The project was developed as part of our Programming Fundamentals course to demonstrate how core programming concepts can be applied to solve a real-world problem.

---

## Features

* User authentication
* Income tracking
* Expense management
* Category-based budgeting
* Savings goals and progress tracking
* Lent and borrowed money management
* Financial summary dashboard
* Built-in financial calculators
* AI assistant ("Mark") for basic financial guidance
* Automatic data persistence using JSON files
* Simple terminal-based interface

---

## Tech Stack

* Python 3
* JSON for data storage
* Gemini API (AI Assistant)
* Terminal/CLI Interface

---

## Modules

### Authentication

* User registration and login
* Email validation
* Secure user-specific data storage

### Income & Budget

* Record income from multiple sources
* Create category-wise budgets
* Update and manage budgets

### Expense Management

* Add, edit, and delete expenses
* Organize expenses into categories
* Automatically calculate total spending

### Savings & Goals

* Create savings goals
* Track progress toward financial targets
* Update savings over time

### Lent & Borrowed

* Keep track of money lent to others
* Record borrowed money
* Monitor outstanding balances

### Financial Calculator

* Perform basic financial calculations
* Estimate future savings and financial growth

### AI Assistant – Mark

* Provides beginner-friendly financial guidance
* Offers spending and saving suggestions
* Uses the Gemini API while avoiding risky or professional financial advice

### Financial Summary

* Displays an overview of:

  * Income
  * Expenses
  * Savings
  * Budgets
  * Debts

### Data Storage

* Automatically saves all data in JSON files
* Loads user data when the application starts

## How It Works

1. Create an account or log in.
2. Record your income and set budgets.
3. Add daily expenses.
4. Track savings and financial goals.
5. Manage debts and borrowed money.
6. View your complete financial summary.
7. Ask Mark for simple financial suggestions.
